# 30hp Drumkit module
![](https://raw.githubusercontent.com/Fihdi/MiniDrumkit/refs/heads/main/MiniDrumkit.jpeg)
Eurorack DrumKit module with an Arduino Nano ESP32 3 channel wave-file player and analog Hi-Hat and Kick Drum sections.

The WAV Files are read from a microSD card and need to be called "1.wav", "2.wav" and "3.wav". The files need to be mono, encoded in 16-bit and have 16kHz sampling rate. You can use Audacity to export wav files in this format.

Demo Video: https://www.youtube.com/watch?v=cPHg9nKNO_o
